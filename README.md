# Terra Grind Cafe - Landing Page


**Landing page modern dan responsif untuk cafe dengan desain earthy tones yang menawan.** Landing page modern dan responsif untuk cafe "Terra Grind Cafe" yang dirancang dengan konsep minimalis dan earthy tones. Mengimplementasikan fitur-fitur interaktif seperti carousel testimoni, form reservasi dengan validasi, dark/light mode toggle, dan animasi counter untuk menciptakan pengalaman pengguna yang optimal.

##  Fitur Utama

## Desain & User Experience
- Palet warna earthy tones (coklat, krem, hijau sage) yang hangat dan inviting
- Layout responsif dengan breakpoints untuk desktop, tablet, dan mobile
- Animasi CSS yang halus dan transisi yang elegan
- Toggle dark/light mode dengan penyimpanan preferensi

## Fitur Interaktif
- Carousel testimoni dengan auto-rotate dan navigasi manual
- Form reservasi dengan validasi real-time
- Animated counters untuk statistik cafe
- Smooth scrolling untuk navigasi antar section
- Mobile-friendly navigation dengan hamburger menu

## Optimasi Performa
- Single HTML file untuk kemudahan deployment
- Gambar optimized dari Unsplash
- Font lokal dengan fallback yang tepat
- JavaScript modular dan terstruktur

## Teknologi Stack
- HTML5 - Semantic markup untuk struktur konten
- CSS3 - Custom properties, flexbox, grid, dan animasi
- JavaScript ES6+ - Vanilla JS tanpa dependency
- Font Awesome 6 - Icon toolkit
- Google Fonts - Poppins & Open Sans
- Unsplash API - High-quality placeholder images

## Cara Menjalankan
1. Clone repository: `git clone https://github.com/ArdhikaTama/LandingPage_Terra-Grind-Cafe.git`
2. Buka file `index.html` di browser modern
3. Atau gunakan live server untuk development
